{% trans "What is Average Return and CAGR?" %}

{% blocktrans %}The average return can mean the simple (arithmetic) average of period returns or the geometric average. The geometric average is the same as the Compound Annual Growth Rate (CAGR): the constant yearly rate that would turn your starting value into your ending value over the same period. CAGR smooths volatility and is the standard way to compare investment performance over time.{% endblocktrans %}

CAGR = (Ending Value ÷ Beginning Value)^(1 ÷ Years) − 1

{% blocktrans %}Use Start/End Values when you only know initial and final amounts; use Annual Returns when you have year-by-year returns and want arithmetic mean, geometric mean, standard deviation, and best/worst year.{% endblocktrans %}

{% trans "Formulas" %}

{% trans "CAGR from start and end values:" %}

{% trans "Geometric mean (equivalent to CAGR) from N period returns:" %}
