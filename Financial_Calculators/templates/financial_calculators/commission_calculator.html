{% trans "What is a Commission Calculator?" %}

{% blocktrans %}A commission calculator helps you figure out how much you or your sales team earn from sales, based on different pay structures. Sales commissions can be a flat percentage of sales, tiered (higher rates at higher sales levels), or combined with a base salary. This tool supports simple percentage commission, tiered brackets, base salary plus commission, and reverse calculation (sales needed to hit a target commission).{% endblocktrans %}

{% trans "Commission Types" %}

{% trans "Simple" %}

{% trans "One flat commission rate applied to the full sales amount. Commission = Sales × Rate%." %}

{% trans "Tiered" %}

{% trans "Different rates for different sales brackets (e.g. 5%% on first $10k, 7%% on next $15k). Total commission is the sum of commission in each tier." %}

{% trans "Base + Commission" %}

{% trans "Fixed base salary plus a percentage of sales. Optionally track quota achievement and a bonus for sales above quota." %}

{% trans "Reverse" %}

{% trans "Enter your target commission and rate to find how much sales you need: Sales Needed = Desired Commission ÷ Rate%." %}

{% trans "How to Use This Calculator" %}

  1. {% trans "Select the commission type (Simple, Tiered, Base + Comm, or Reverse)." %}
  2. {% trans "Enter the requested amounts: sales, rate(s), and for Base + Comm: base salary and optional quota." %}
  3. {% trans "For Tiered, use the default tiers or leave as is; sales amount is the only input." %}
  4. {% trans "Click Calculate Commission to see the main result, breakdown, and any chart." %}
