{% trans "Complete User Guide" %}

{% trans "What is the Currency Calculator?" %}

{% blocktrans %}This calculator converts an amount from one currency to another using exchange rates. You enter the amount and select the source and target currencies. The result shows the converted amount, the exchange rate (how many units of the target currency you get for one unit of the source), and the inverse rate. A table of common amounts and cross rates for major currencies helps you quickly see equivalent values.{% endblocktrans %}

{% trans "Key Concepts" %}

{% trans "Exchange rate" %}

{% trans "The rate at which one currency is exchanged for another. For example, 1 USD = 0.92 EUR means one US dollar buys 0.92 euros." %}

{% trans "Base currency" %}

{% trans "Rates here are quoted relative to USD. Conversion is done by first converting to USD, then to the target currency." %}

{% trans "Cross rates" %}

{% trans "The same amount converted into several major currencies (USD, EUR, GBP, JPY, CAD, AUD, CHF) so you can compare equivalents at a glance." %}

{% trans "How to Use This Calculator" %}

  1. {% trans "Enter the amount you want to convert (e.g. 1000)." %}
  2. {% trans "Select the source currency (From) and the target currency (To). Use the swap button to reverse them." %}
  3. {% trans "Click Convert to see the result, exchange rate, common amounts table, and cross rates." %}
  4. {% trans "Use the chart to compare your amount in major currencies." %}
  5. {% trans "Rates are indicative; actual rates may vary at banks or exchange services." %}

{% trans "Understanding Your Results" %}

{% trans "Converted amount" %}

{% trans "The equivalent amount in the target currency at the given exchange rate." %}

{% trans "Exchange rate & inverse" %}

{% trans "Rate: 1 From = X To. Inverse: 1 To = Y From. Useful for quick mental conversion in both directions." %}

{% trans "Common amounts & cross rates" %}

{% trans "Common amounts shows 1, 5, 10, 25, 50, 100, 500, 1000 in the from currency and their equivalents. Cross rates show your amount in USD, EUR, GBP, JPY, CAD, AUD, CHF (excluding the from currency)." %}

{% trans "Understanding the Chart" %}

{% trans "The bar chart shows your entered amount converted into each of the other major currencies (USD, EUR, GBP, JPY, CAD, AUD, CHF). This lets you compare purchasing power across currencies at a glance." %}
