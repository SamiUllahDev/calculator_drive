{% trans "What is the Depreciation Calculator?" %}

{% blocktrans %}This calculator computes depreciation for an asset using several common methods: straight-line (equal amount each year), declining balance, double declining balance, sum-of-years' digits, units of production, and MACRS (US tax). You enter cost, salvage value, and useful life; it returns a year-by-year schedule, total depreciation, and charts.{% endblocktrans %}

{% trans "Methods" %}

  • {% trans "Straight-line:" %} {% trans "Depreciable amount ÷ useful life. Same amount each year." %}
  • {% trans "Declining balance:" %} {% trans "Apply a fixed %% to the remaining book value each year. Rate = (e.g. 150%% or 200%%) ÷ useful life." %}
  • {% trans "Double declining:" %} {% trans "200%% declining balance: rate = 2 ÷ useful life. Stops at salvage value." %}
  • {% trans "Sum-of-years' digits:" %} {% trans "Depreciable amount × (remaining life / sum of years). Front-loaded." %}
  • {% trans "Units of production:" %} {% trans "Depreciation per unit = depreciable ÷ total units; each year = units produced × per unit. Requires total units." %}
  • MACRS: {% trans "US tax method. Use useful life 3, 5, 7, 10, or 15. Ignores salvage value." %}

  • {% trans "Salvage value must be less than asset cost (except for MACRS, which ignores it)." %}
  • {% trans "MACRS property classes: 3, 5, 7, 10, 15 years. Half-year convention is assumed." %}
  • {% trans "For units of production, if you don't provide units per year, equal distribution over useful life is assumed." %}
