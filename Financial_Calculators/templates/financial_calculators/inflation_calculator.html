{% trans "What is Inflation?" %}

{% blocktrans %}Inflation is the rate at which the general level of prices for goods and services rises over time, eroding purchasing power. It means that each unit of currency buys fewer goods and services as time goes on. Inflation is measured as an annual percentage increase.{% endblocktrans %}

{% trans "Future Value = Present Value × (1 + r)ⁿ" %}

{% blocktrans %}Where r is the annual inflation rate (as a decimal) and n is the number of years. This formula shows how much money you would need in the future to have the same purchasing power as a given amount today.{% endblocktrans %}

{% trans "Inflation Categories" %}

{% trans "Deflation: Rate < 0%" %}

{% trans "Prices are falling. While consumers benefit short-term, prolonged deflation can harm the economy by discouraging spending and investment." %}

{% trans "Low to Moderate: Rate 0% - 4%" %}

{% trans "Healthy inflation range. Most central banks target around 2% annual inflation, which encourages spending and investment while keeping prices relatively stable." %}

{% trans "High: Rate 4% - 7%" %}

{% trans "Above-target inflation. Purchasing power erodes noticeably. Savings in low-yield accounts lose real value. Consider inflation-protected investments." %}

{% trans "Very High: Rate 7% - 10%" %}

{% trans "Significantly elevated inflation. Money loses value rapidly. Essential costs increase substantially. Diversified investment strategies become critical." %}

{% trans "Extreme / Hyperinflation: Rate > 10%" %}

{% trans "Extremely high inflation that can destabilize economies. Historical examples include Zimbabwe (2008) and Venezuela (2018+). Asset protection becomes paramount." %}

{% trans "Understanding Your Results" %}

{% trans "Future Cost / Today's Value" %}

{% trans "In Future Cost mode: the amount you'll need in the future to match today's purchasing power. In Past Value mode: what a past amount would be worth in today's dollars." %}

{% trans "Purchasing Power Lost" %}

{% trans "The dollar amount of purchasing power eroded by inflation over the time period. This is the difference between the nominal value and the real value." %}

{% trans "Cumulative Inflation" %}

{% trans "The total percentage increase in prices over the entire period. For example, at 3% annual inflation over 10 years, cumulative inflation is about 34.4%." %}

{% trans "Real Value" %}

{% trans "The value of your money adjusted for inflation. This shows what your money is actually worth in today's purchasing power after accounting for price increases." %}

{% trans "Inflation Formulas" %}

{% trans "The equations used for calculating the effect of inflation:" %}

{% trans "Future Cost:" %}

FV = PV × (1 + r)ⁿ

{% trans "Real Value (Purchasing Power):" %}

RV = FV / (1 + r)ⁿ

{% trans "Cumulative Inflation:" %}

CI = ((1 + r)ⁿ - 1) × 100%
