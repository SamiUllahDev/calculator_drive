{% trans "Complete User Guide" %}

{% trans "What is the Inflation Calculator?" %}

{% blocktrans %}This calculator shows how inflation affects the value of money over time. You enter an amount, an annual inflation rate, and a number of years. In "Future Cost" mode it tells you how much money you would need in the future to have the same purchasing power as today. In "Past Value" mode it tells you what a past amount would be worth in today's terms. It also shows cumulative inflation and a comparison at different inflation rates.{% endblocktrans %}

{% trans "Key Concepts" %}

{% trans "Inflation" %}

{% trans "The rate at which prices rise over time. A 3% annual inflation rate means that on average, something that costs $100 today will cost $103 in one year." %}

{% trans "Purchasing power" %}

{% trans "How much your money can buy. As inflation rises, the same amount of money buys less—so purchasing power falls." %}

{% trans "Real value" %}

{% trans "The value of money adjusted for inflation. Real value shows what your money is worth in today's terms after accounting for price increases." %}

{% trans "How to Use This Calculator" %}

  1. {% trans "Choose Future Cost (how much you'll need later) or Past Value (what past money is worth today)." %}
  2. {% trans "Enter the amount in dollars and the annual inflation rate (e.g. 3 for 3%%)." %}
  3. {% trans "Enter the time period in years. Click Calculate to see the result and charts." %}
  4. {% trans "Use the rate comparison table and chart to see how different inflation rates change the outcome." %}
  5. {% trans "Review the yearly breakdown to see value year by year." %}

{% trans "Understanding Your Results" %}

{% trans "Future cost / Today's value" %}

{% trans "In Future Cost mode: the amount needed in the future to match today's purchasing power. In Past Value mode: the equivalent of the past amount in today's dollars." %}

{% trans "Purchasing power lost" %}

{% trans "The difference between the nominal (face) value and the real value—how much inflation has eroded." %}

{% trans "Cumulative inflation" %}

{% trans "Total percentage increase in prices over the period. For example, 34.4% over 10 years at 3% per year." %}

{% trans "Understanding the Charts" %}

{% trans "Value Over Time" %}

{% trans "Line chart showing nominal value (what the amount grows to or from) and real value (purchasing power) year by year. The gap between the two lines shows the effect of inflation." %}

{% trans "Compare Inflation Rates" %}

{% trans "Bar chart showing future cost at 2%%, 3%%, 4%%, 5%%, 7%%, and 10%% inflation. Helps you see how sensitive the result is to the rate you assume." %}

{% trans "Important Notes" %}

  • {% trans "Inflation varies by year and by category. This uses a single average rate for the whole period." %}
  • {% trans "To protect against inflation, consider investments that historically outpace it: stocks, real estate, I-bonds, TIPS." %}
  • {% trans "US historical average is around 3%% per year; actual rates can be higher or lower." %}
