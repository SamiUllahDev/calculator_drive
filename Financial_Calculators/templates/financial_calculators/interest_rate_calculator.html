{% trans "What is APR vs APY?" %}

{% blocktrans %}APR (Annual Percentage Rate) is the nominal rate before compounding. APY (Annual Percentage Yield) is the actual return after compounding. When interest compounds more than once per year, APY is higher than APR. This calculator finds APR from payment and term, converts between APR and APY, and shows effective rate when fees are included.{% endblocktrans %}

{% trans "Calculation Types" %}

  • {% trans "Find Rate" %}: {% trans "Enter loan amount, monthly payment, and term to get the implied APR." %}
  • {% trans "APR → APY" %}: {% trans "Convert a stated APR to APY for a given compounding frequency." %}
  • {% trans "APY → APR" %}: {% trans "Convert APY back to the equivalent APR." %}
  • {% trans "Effective Rate" %}: {% trans "Include origination fees, points, and other fees to see the true cost of the loan (effective annual rate)." %}
  • {% trans "Compare Rates" %}: {% trans "Enter multiple rates to compare monthly payments and total interest." %}

{% trans "Formulas Used" %}

APY = (1 + APR/n)^n − 1

APR = n × [(1 + APY)^(1/n) − 1]

{% trans "n = compounding periods per year. Find rate uses the loan payment formula solved for rate (Newton-Raphson)." %}
