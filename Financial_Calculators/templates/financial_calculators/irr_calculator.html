{% trans "What is IRR?" %}

{% blocktrans %}The Internal Rate of Return (IRR) is the discount rate that makes the Net Present Value (NPV) of all cash flows from a project equal to zero. It represents the expected annual return of the investment. If IRR is greater than your required return (cost of capital), the project is typically considered acceptable.{% endblocktrans %}

NPV = 0 = CF₀ + CF₁/(1+IRR) + CF₂/(1+IRR)² + …
