{% trans "What is the Margin Calculator?" %}

{% blocktrans %}This calculator helps you work with profit margin and markup. You can find margin and markup from cost and selling price, find the selling price from cost and a desired margin or markup, find cost from price and margin, or convert between margin and markup. Margin is profit as a percentage of selling price; markup is profit as a percentage of cost. The tool includes a comparison table and charts.{% endblocktrans %}

{% trans "Key Concepts" %}

{% trans "Margin" %}

{% trans "Profit as a percentage of selling price. Margin = (Profit / Price) × 100. Used when you want to know what share of revenue is profit." %}

{% trans "Markup" %}

{% trans "Profit as a percentage of cost. Markup = (Profit / Cost) × 100. Common in retail: e.g. 50% markup means you add half the cost to set the price." %}

{% trans "Difference" %}

{% trans "For the same deal, margin is always a smaller percentage than markup. Example: 50% margin = 100% markup (price is double the cost)." %}
