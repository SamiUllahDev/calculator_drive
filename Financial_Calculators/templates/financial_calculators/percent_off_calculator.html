{% trans "What is the Percent Off Calculator?" %}

{% blocktrans %}This calculator handles several percent-off scenarios: find the sale price from an original price and a discount %%; find the discount %% from original and sale price; find the original price from sale price and %%; apply multiple stacked discounts; or model buy-X-get-Y deals (e.g. buy 2 get 1 free). It shows what you pay, what you save, and includes comparison charts.{% endblocktrans %}

{% trans "Modes" %}

  • {% trans "% off from two prices:" %} {% trans "Enter original and sale price to get the equivalent percent off and savings." %}
  • {% trans "Original from sale + % off:" %} {% trans "Enter sale price and percent off to work out the original price." %}
  • {% trans "Stacked discounts:" %} {% trans "Enter original price and a list of discount percents (e.g. 20, 10). Each discount applies to the previous result. See effective total %." %}
  • {% trans "Buy X Get Y:" %} {% trans "Enter price per item, buy quantity, get quantity, and get discount %% (100 = free). See deal total and effective discount." %}
