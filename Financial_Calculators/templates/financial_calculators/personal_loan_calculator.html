{% trans "Complete User Guide" %}

{% trans "What is the Personal Loan Calculator?" %}

{% trans "This calculator helps you estimate personal loan payments, see how much you can borrow for a target monthly payment, or compare different loan terms. Personal loans are unsecured and can be used for debt consolidation, home improvements, medical bills, or other expenses." %}

{% trans "You can optionally include an origination fee (a percentage of the loan amount some lenders charge). Results include monthly payment, total interest, total cost, APR, payment breakdown and balance charts, and an amortization schedule." %}

{% trans "Key Metrics" %}

{% trans "Monthly Payment" %}

{% trans "The fixed amount you pay each month. Shorter terms mean higher payments but less total interest." %}

{% trans "Total Interest & Total Cost" %}

{% trans "Total interest is the cost of borrowing. Total cost adds any origination fee so you see the full cost of the loan." %}

{% trans "APR" %}

{% trans "Annual Percentage Rate includes interest and fees, giving you a true cost to compare across lenders." %}

{% trans "Net Proceeds" %}

{% trans "Loan amount minus origination fee: the cash you actually receive. Payments are based on the full loan amount." %}

{% trans "How to Use This Calculator" %}

  1. {% trans "Choose a mode: Calculate Payment, How Much Can I Borrow, or Compare Loan Terms." %}
  2. {% trans "Enter loan amount (or desired payment for affordability), interest rate (APR), and term." %}
  3. {% trans "Optionally enter an origination fee percentage." %}
  4. {% trans "Click Calculate. View monthly payment, total interest, charts, and amortization (payment mode) or comparison table and chart (compare mode)." %}
  5. {% trans "In payment results, switch between Monthly and Yearly schedule views." %}

{% trans "Understanding Your Results" %}

{% trans "Calculate Payment" %}

{% trans "Shows your fixed monthly payment, total interest, total cost (interest + fee), and APR. The amortization schedule shows how each payment splits between principal and interest and the remaining balance." %}

{% trans "How Much Can I Borrow?" %}

{% trans "Enter the monthly payment you can afford; the calculator shows the maximum loan amount you can get at the given rate and term." %}

{% trans "Compare Loan Terms" %}

{% trans "Shows monthly payment and total interest for 12–84 months so you can see how term length affects payment size and total cost." %}

{% trans "Understanding the Charts" %}

{% trans "Payment breakdown:" %} {% trans "Doughnut chart of principal vs. interest so you see how much of your total payments go to the loan vs. borrowing cost." %}

{% trans "Balance over time:" %} {% trans "Line chart of remaining balance by month. In Compare Terms mode, a bar chart shows total interest by term length." %}

{% trans "Formulas" %}

{% trans "Monthly Payment (M)" %}

M = P × [r(1+r)^n] / [(1+r)^n − 1]

{% trans "P = loan amount, r = monthly rate (APR ÷ 12), n = number of months" %}

{% trans "Origination fee" %}

{% trans "Fee = Loan amount × (Origination fee % ÷ 100). Net proceeds = Loan amount − Fee." %}
