{% trans "What is Present Value?" %}

{% blocktrans %}Present value (PV) is what a future sum of money is worth today, given a discount rate. Money today is worth more than the same amount in the future because you can invest it and earn returns. PV is used to compare investments, value bonds, and decide between a lump sum and payments over time.{% endblocktrans %}

{% trans "What Your Results Mean" %}

{% trans "Present Value" %}

{% trans "The amount you would need to invest today (at the given rate) to reach the future value at the end of the period." %}

{% trans "Discount Factor" %}

{% trans "PV divided by FV; how much one dollar in the future is worth today (0 to 1). Lower for longer periods or higher rates." %}
