{% trans "Complete User Guide" %}

{% trans "What is BMI?" %}

{% trans "Body Mass Index (BMI) is a measure calculated from your weight and height. It's used as a screening tool to identify potential weight-related health issues. The formula is:" %}

BMI = Weight (kg) ÷ Height (m)²

{% trans "While BMI is a useful screening tool, it doesn't directly measure body fat or account for muscle mass, bone density, or other factors. For very low BMI ranges, it's particularly important to consult healthcare professionals." %}

{% trans "Anorexia-Related BMI Ranges" %}

{% trans "Severe Anorexia: BMI < 15" %}

{% trans "This indicates severe underweight status. Immediate medical attention is strongly recommended." %}

{% trans "Anorexia: BMI 15-17.5" %}

{% trans "This range falls below the DSM-5 threshold for anorexia nervosa (BMI < 17.5). Professional evaluation is recommended." %}

{% trans "Underweight: BMI 17.5-18.5" %}

{% trans "Below the normal weight range. Consider consulting a healthcare provider to ensure adequate nutrition." %}

{% trans "Normal: BMI ≥ 18.5" %}

{% trans "Within the healthy weight range. Continue maintaining a balanced diet and regular physical activity." %}

{% trans "Important Notes" %}

  • {% trans "This calculator is for educational and informational purposes only. It is not a diagnostic tool." %}
  • {% trans "BMI does not account for muscle mass, bone density, or body composition. Athletes or individuals with high muscle mass may have higher BMIs that don't reflect excess fat." %}

{% trans "When to Seek Professional Help" %}

{% trans "Consider consulting a healthcare professional if:" %}

  • {% trans "Your BMI is below 17.5 (anorexic range)" %}
  • {% trans "You're experiencing rapid weight loss" %}
  • {% trans "You have concerns about your eating patterns" %}
  • {% trans "You're experiencing physical symptoms like fatigue, dizziness, or irregular periods" %}
  • {% trans "You have persistent thoughts about food, weight, or body image that interfere with daily life" %}
