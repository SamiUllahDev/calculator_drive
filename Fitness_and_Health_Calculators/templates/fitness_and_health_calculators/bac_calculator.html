{% trans "Complete User Guide" %}

{% trans "What is Blood Alcohol Content (BAC)?" %}

{% trans "Blood Alcohol Content (BAC) is a measure of the amount of alcohol in your bloodstream, expressed as a percentage. It's used to determine legal intoxication levels and assess impairment." %}

{% trans "BAC is measured in grams of alcohol per 100 milliliters of blood. For example, a BAC of 0.08% means 0.08 grams of alcohol per 100 milliliters of blood." %}

{% trans "Legal BAC Limits" %}

{% trans "United States" %}

{% trans "Legal limit: 0.08% BAC for drivers 21 and over. Zero tolerance (0.00-0.02%) for drivers under 21." %}

{% trans "Canada" %}

{% trans "Legal limit: 0.08% BAC (0.05% in some provinces). Zero tolerance for drivers under 21 or with learner's permits." %}

{% trans "United Kingdom" %}

{% trans "Legal limit: 0.08% BAC in England, Wales, and Northern Ireland. 0.05% in Scotland." %}

{% trans "BAC Effects by Level" %}

{% trans "0.00% - 0.02%: No Effects" %}

{% trans "No noticeable effects. Safe to drive (if legally allowed)." %}

{% trans "0.02% - 0.05%: Mild Effects" %}

{% trans "Slight relaxation, mild mood elevation. Judgment and coordination may be slightly impaired." %}

{% trans "0.05% - 0.08%: Impaired" %}

{% trans "Reduced coordination, impaired judgment, decreased alertness. NOT safe to drive." %}

{% trans "0.08% - 0.15%: Significantly Impaired" %}

{% trans "Poor muscle coordination, balance problems, significantly impaired judgment. Illegal to drive." %}

{% trans "0.15%+: Severely Impaired" %}

{% trans "Severe motor impairment, loss of physical control, possible blackouts. Medical attention may be needed." %}

{% trans "What is a Standard Drink?" %}

{% trans "A standard drink contains approximately 14 grams (0.6 ounces) of pure alcohol. Examples:" %}

• {% trans "Beer: 12 ounces (355 ml) of regular beer (5% ABV)" %}
• {% trans "Wine: 5 ounces (148 ml) of wine (12% ABV)" %}
• {% trans "Liquor: 1.5 ounces (44 ml) of distilled spirits (40% ABV)" %}

{% trans "Note: Actual alcohol content varies by brand and type. This calculator uses average values." %}

{% trans "Widmark Formula" %}

{% trans "This calculator uses the Widmark formula to estimate BAC:" %}

{% trans "BAC = (Alcohol in grams / (Weight in kg × r)) - (β × Hours)" %}

r: {% trans "Distribution ratio (0.68 for men, 0.55 for women)" %}

β: {% trans "Elimination rate (0.15 g/L per hour average)" %}

{% trans "The formula accounts for alcohol absorption, distribution in the body, and elimination over time." %}

{% trans "⚠️ Important Safety Warnings" %}

  • {% trans "This calculator provides ESTIMATES only. Actual BAC can vary significantly based on metabolism, food intake, medication, and other factors." %}
  • {% trans "NEVER drink and drive. If you've consumed any alcohol, arrange for a designated driver, use public transportation, or call a taxi/ride-share service." %}
  • {% trans "Only time can lower BAC. Coffee, cold showers, food, or exercise do NOT reduce BAC or make you sober." %}

{% trans "Body Weight" %}: {% trans "Heavier people generally have lower BAC for the same amount of alcohol." %}

{% trans "Gender" %}: {% trans "Women typically have higher BAC than men for the same amount of alcohol due to body composition differences." %}

{% trans "Food" %}: {% trans "Eating before or while drinking slows alcohol absorption." %}

{% trans "Metabolism" %}: {% trans "Individual metabolic rates vary, affecting how quickly alcohol is processed." %}

{% trans "Medications" %}: {% trans "Some medications can interact with alcohol and increase impairment." %}
