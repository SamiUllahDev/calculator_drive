{% trans "What is Body Surface Area (BSA)?" %}

{% trans "Body Surface Area (BSA) is the total surface area of the human body. It's measured in square meters (m²) and is used in various medical applications, particularly for calculating drug dosages, determining cardiac index, and assessing metabolic rate." %}

{% blocktrans trimmed %}BSA provides a more accurate measure than body weight alone for many medical calculations, as it accounts for both height and weight, giving a better representation of body size. For other body metrics, also check your{% endblocktrans %} {% trans "BMI" %} {% trans "and" %} {% trans "Body Fat Percentage" %}.

{% trans "BSA Calculation Formulas" %}

{% trans "Du Bois Formula (1916)" %}

{% trans "One of the oldest and most widely used formulas. Considered the gold standard." %}

{% trans "BSA = 0.007184 × weight^0.425 × height^0.725" %}

{% trans "Where weight is in kg and height is in cm" %}

{% trans "Mosteller Formula (1987)" %}

{% trans "Simplified formula, easy to remember and calculate. Commonly used in clinical practice." %}

{% trans "BSA = √((height × weight) / 3600)" %}

{% trans "Where height is in cm and weight is in kg" %}

{% trans "Haycock Formula (1978)" %}

{% trans "Developed specifically for pediatric populations but also accurate for adults." %}

{% trans "BSA = 0.024265 × weight^0.5378 × height^0.3964" %}

{% trans "Where weight is in kg and height is in cm" %}

{% trans "Gehan & George Formula (1970)" %}

{% trans "Another commonly used formula, particularly in oncology and chemotherapy dosing." %}

{% trans "BSA = 0.0235 × weight^0.51456 × height^0.42246" %}

{% trans "Where weight is in kg and height is in cm" %}

{% trans "Medical Applications of BSA" %}

{% trans "Drug Dosage Calculation:" %} {% trans "Many medications, especially chemotherapy drugs, are dosed based on BSA rather than body weight to ensure appropriate drug levels." %}

{% trans "Cardiac Index:" %} {% trans "BSA is used to calculate cardiac index (cardiac output divided by BSA), which helps assess heart function." %}

{% trans "Renal Function:" %} {% blocktrans trimmed %}Glomerular filtration rate (GFR) is often normalized to BSA for more accurate assessment of kidney function. Estimate yours with our{% endblocktrans %} {% trans "GFR Calculator" %}.

{% trans "Fluid Requirements:" %} {% trans "Some fluid replacement calculations use BSA to determine appropriate volumes." %}
