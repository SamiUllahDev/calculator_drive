{% trans "What are Body Types?" %}

{% trans "Body types, also known as somatotypes, are classifications of human body shapes based on skeletal frame size and body composition. The three main body types are ectomorph, mesomorph, and endomorph, originally proposed by psychologist William Sheldon in the 1940s." %}

{% trans "While everyone is unique, understanding your body type can help you make informed decisions about nutrition, exercise, and fitness goals that work best with your natural body structure." %}

{% trans "The Three Body Types" %}

{% trans "Ectomorph" %}

{% trans "Small frame, naturally lean, fast metabolism" %}

  • {% trans "Naturally thin and lean build" %}
  • {% trans "Fast metabolism" %}
  • {% trans "Difficulty gaining weight" %}
  • {% trans "Narrow shoulders and hips" %}
  • {% trans "Long limbs" %}
  • {% trans "Low body fat percentage" %}

{% trans "Mesomorph" %}

{% trans "Medium frame, athletic build, moderate metabolism" %}

  • {% trans "Athletic and muscular build" %}
  • {% trans "Moderate metabolism" %}
  • {% trans "Gains muscle easily" %}
  • {% trans "Broad shoulders" %}
  • {% trans "Narrow waist" %}
  • {% trans "Naturally strong" %}

{% trans "Endomorph" %}

{% trans "Large frame, wider build, slower metabolism" %}

  • {% trans "Naturally wider frame" %}
  • {% trans "Slower metabolism" %}
  • {% trans "Stores fat easily" %}
  • {% trans "Wider hips and shoulders" %}
  • {% trans "Gains muscle and fat easily" %}
  • {% trans "Rounder body shape" %}

{% trans "How Frame Size is Calculated" %}

{% trans "This calculator uses the wrist-to-height ratio method to determine frame size:" %}

{% trans "Wrist-to-Height Ratio = (Wrist Circumference / Height) × 100" %}

{% trans "Small Frame:" %} {% trans "Wrist-to-height ratio < 9.9%" %}

{% trans "Medium Frame:" %} {% trans "Wrist-to-height ratio 9.9% - 10.9%" %}

{% trans "Large Frame:" %} {% trans "Wrist-to-height ratio > 10.9%" %}

{% trans "Frame size is then used to determine body type: Small = Ectomorph, Medium = Mesomorph, Large = Endomorph." %}

{% trans "How to Take Measurements" %}

{% trans "Wrist Measurement" %}

{% trans "Measure around your wrist at the narrowest point, just above the wrist bone. Keep the tape measure snug but not tight. This measurement is used to determine frame size." %}

{% trans "Ankle Measurement" %}

{% trans "Measure around your ankle at the narrowest point, just above the ankle bone. Keep the tape measure level and snug but not tight." %}

{% trans "Height Measurement" %}

{% trans "Stand straight against a wall without shoes. Measure from the floor to the top of your head. For best accuracy, measure in the morning when you're at your tallest." %}

{% trans "Tip:" %} {% trans "For best accuracy, take measurements in the morning before eating, and measure multiple times to ensure consistency." %}
