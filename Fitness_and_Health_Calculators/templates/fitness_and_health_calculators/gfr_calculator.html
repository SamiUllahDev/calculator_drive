{% trans "Complete User Guide" %}

{% trans "What is GFR?" %}

{% trans "GFR (Glomerular Filtration Rate) is a measure of how well your kidneys are filtering waste from your blood. It represents the rate at which fluid is filtered through the glomeruli (tiny filters in the kidneys) per unit of time." %}

{% trans "eGFR (estimated GFR) is calculated using formulas that take into account your creatinine level, age, gender, and race. It's expressed in milliliters per minute per 1.73 square meters of body surface area (mL/min/1.73m²)." %}

{% trans "Why GFR is Important" %}

{% trans "GFR is the best overall indicator of kidney function. It helps healthcare providers:" %}

  • {% trans "Detect kidney disease early" %}
  • {% trans "Monitor kidney function over time" %}
  • {% trans "Determine the stage of chronic kidney disease (CKD)" %}
  • {% trans "Make decisions about treatment and medication dosing" %}
  • {% trans "Assess the risk of kidney disease progression" %}

{% trans "eGFR Value" %}

{% trans "Your estimated glomerular filtration rate in mL/min/1.73m². Higher values indicate better kidney function. Normal eGFR is typically 90 or higher." %}

{% trans "CKD Stage" %}

{% trans "Your chronic kidney disease stage based on your eGFR value. Stages range from Stage 1 (normal/high) to Stage 5 (kidney failure)." %}

{% trans "CKD-EPI vs MDRD" %}

{% trans "The calculator shows results from both formulas. CKD-EPI is the preferred formula as it's more accurate, especially at higher eGFR levels. MDRD is shown for comparison." %}

{% trans "Chronic Kidney Disease Stages Explained" %}

{% trans "Stage 1 (eGFR ≥ 90)" %}

{% trans "Normal or high kidney function. May have kidney damage but function is preserved. Regular monitoring recommended." %}

{% trans "Stage 2 (eGFR 60-89)" %}

{% trans "Mildly decreased kidney function. Kidney damage with mild loss of function. Monitoring and lifestyle changes recommended." %}

{% trans "Stage 3a (eGFR 45-59)" %}

{% trans "Mildly to moderately decreased kidney function. More frequent monitoring and may need to see a nephrologist." %}

{% trans "Stage 3b (eGFR 30-44)" %}

{% trans "Moderately to severely decreased kidney function. Regular nephrologist visits and treatment planning recommended." %}

{% trans "Stage 4 (eGFR 15-29)" %}

{% trans "Severely decreased kidney function. Preparation for kidney replacement therapy (dialysis or transplant) should begin." %}

{% trans "Stage 5 (eGFR < 15)" %}

{% trans "Kidney failure. Dialysis or kidney transplant is typically needed. Also called end-stage renal disease (ESRD)." %}

{% trans "eGFR Formulas Explained" %}

{% trans "CKD-EPI Formula" %}

{% trans "The preferred formula for calculating eGFR. More accurate than MDRD, especially at higher eGFR levels (above 60). Recommended by most medical organizations." %}

{% trans "Takes into account: Creatinine, age, gender, and race" %}

{% trans "MDRD Formula" %}

{% trans "An older formula still used for comparison. Less accurate at higher eGFR levels but useful for historical comparison." %}

{% trans "Takes into account: Creatinine, age, gender, and race" %}

{% trans "Factors Affecting GFR" %}

{% trans "Age:" %} {% trans "GFR naturally decreases with age. Normal decline is about 1 mL/min/1.73m² per year after age 40." %}

{% trans "Gender:" %} {% trans "Men typically have higher muscle mass and creatinine levels, which affects the calculation." %}

{% trans "Race:" %} {% trans "The formulas include race adjustments based on research showing differences in muscle mass and creatinine production." %}

{% trans "Creatinine Level:" %} {% trans "Higher creatinine levels indicate reduced kidney function, as kidneys are less able to filter waste from the blood." %}

{% trans "Muscle Mass:" %} {% trans "People with higher muscle mass produce more creatinine, which can affect eGFR calculations." %}

{% trans "Important Notes" %}

  • {% trans "eGFR is an estimate. Actual GFR can only be measured directly through specialized tests. The calculator provides estimates based on standard formulas." %}
  • {% trans "eGFR may not be accurate for certain populations, including children, pregnant women, people with very high or low muscle mass, or those with rapidly changing kidney function." %}
  • {% trans "A single eGFR measurement may not be diagnostic. Healthcare providers typically look at trends over time and consider other factors like protein in urine." %}
  • {% trans "This calculator is for educational and informational purposes only. Always consult with healthcare providers for accurate kidney function assessment, diagnosis, and treatment decisions." %}
