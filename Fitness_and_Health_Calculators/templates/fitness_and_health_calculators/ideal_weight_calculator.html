{% trans "What is Ideal Weight?" %}

{% blocktrans trimmed %}Ideal body weight (IBW) is a target weight range estimated using established medical formulas. It represents the weight associated with optimal health outcomes for your height and gender. For the broader healthy weight range based on BMI categories, also try our{% endblocktrans %} {% trans "Healthy Weight Calculator" %}.

{% blocktrans trimmed %}This calculator uses four well-known formulas — Robinson, Miller, Devine, and Hamwi — each developed from clinical research. The average of these formulas provides a balanced target. To see where your current weight falls on the BMI scale, use our{% endblocktrans %} {% trans "BMI Calculator" %}.

{% trans "Formulas Explained" %}

{% trans "Robinson Formula (1983)" %}

{% trans "One of the most widely referenced formulas. Uses height in inches above 5 feet with different base weights for men and women." %}

{% trans "Miller Formula (1983)" %}

{% trans "Generally produces slightly higher ideal weight estimates. Useful as an upper reference point." %}

{% trans "Devine Formula (1974)" %}

{% trans "Originally developed for medication dosing, this formula became widely adopted for general ideal weight estimation." %}

{% trans "Hamwi Formula (1964)" %}

{% trans "The oldest of the four formulas, still commonly used in clinical practice for quick ideal weight estimation." %}
