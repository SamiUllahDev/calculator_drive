{% trans "What is One Rep Max (1RM)?" %}

{% trans "One Rep Max (1RM) is the maximum amount of weight you can lift for a single repetition of a given exercise. It's a fundamental metric in strength training used to:" %}

  • {% trans "Determine training intensities (percentages of 1RM)" %}
  • {% trans "Track strength progress over time" %}
  • {% trans "Design periodized training programs" %}
  • {% trans "Compare strength levels" %}

{% blocktrans trimmed %}Since testing your true 1RM can be dangerous and requires proper preparation, calculators use formulas to estimate it based on submaximal lifts (lighter weights for multiple reps). To fuel your training properly, check your energy needs with our{% endblocktrans %} {% trans "TDEE Calculator" %} {% trans "and plan your nutrition with our" %} {% trans "Protein Calculator" %}.

{% trans "1RM Calculation Formulas" %}

{% trans "Brzycki Formula" %}

1RM = Weight / (1.0278 - 0.0278 × Reps)

{% trans "Most popular formula. Generally accurate for 1-10 reps." %}

{% trans "Epley Formula" %}

1RM = Weight × (1 + Reps/30)

{% trans "Simple and widely used. Good for moderate rep ranges." %}

{% trans "Lombardi Formula" %}

1RM = Weight × Reps^0.10

{% trans "Power-based formula. Works well for higher rep ranges." %}

{% trans "O'Conner Formula" %}

1RM = Weight × (1 + Reps/40)

{% trans "Conservative estimate. Good for beginners." %}

{% trans "Mayhew Formula" %}

1RM = (Weight × 100) / (52.2 + 41.9 × e^(-0.055 × Reps))

{% trans "Exponential formula. Accurate across wide rep ranges." %}

{% trans "Wathan Formula" %}

1RM = (Weight × 100) / (48.8 + 53.8 × e^(-0.075 × Reps))

{% trans "Another exponential formula. Good alternative to Mayhew." %}

{% trans "The calculator uses the average of all formulas for the most accurate estimate. Different formulas may give slightly different results, which is why using multiple formulas and averaging provides the best estimate." %}
