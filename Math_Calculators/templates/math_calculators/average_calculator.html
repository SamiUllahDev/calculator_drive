{% trans "Average Types Explained" %}

{% trans "Arithmetic Mean" %}

{% trans "Sum of all numbers divided by the count. Most commonly used average." %}

Formula: (x₁ + x₂ + ... + xₙ) / n

{% trans "Median" %}

{% trans "Middle value when numbers are sorted. Less affected by outliers." %}

{% trans "Mode" %}

{% trans "Most frequently occurring value(s). May have multiple modes or none." %}

{% trans "Geometric Mean" %}

{% trans "nth root of the product of all numbers. Useful for rates and ratios." %}

Formula: ⁿ√(x₁ × x₂ × ... × xₙ)

{% trans "Harmonic Mean" %}

{% trans "Reciprocal of the arithmetic mean of reciprocals. Used for rates." %}

Formula: n / (1/x₁ + 1/x₂ + ... + 1/xₙ)

{% trans "Quadratic Mean (RMS)" %}

{% trans "Square root of the mean of squares. Used in physics and engineering." %}

Formula: √((x₁² + x₂² + ... + xₙ²) / n)

  • {% trans "Geometric mean requires all numbers to be positive" %}
  • {% trans "Harmonic mean requires all numbers to be non-zero" %}
  • {% trans "Mode may not exist if all values are unique" %}
