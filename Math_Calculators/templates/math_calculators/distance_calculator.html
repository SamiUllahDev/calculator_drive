{% trans "Understanding Distance Types" %}

{% trans "Euclidean Distance" %}

{% trans "The straight-line distance between two points. In 2D: √((x₂-x₁)² + (y₂-y₁)²). Represents the shortest path." %}

{% trans "Manhattan Distance" %}

{% trans "The sum of absolute differences: |x₂-x₁| + |y₂-y₁|. Also called taxicab distance, representing distance along grid lines." %}

{% trans "Geographic Distance" %}

{% trans "Uses the Haversine formula to calculate great-circle distance between two points on Earth using latitude and longitude." %}

{% trans "Minkowski Distance" %}

{% trans "A generalized distance metric. p=1 gives Manhattan, p=2 gives Euclidean, p=∞ gives Chebyshev distance." %}
