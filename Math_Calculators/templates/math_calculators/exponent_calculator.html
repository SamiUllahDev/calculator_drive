{% trans "Understanding Exponents and Roots" %}

{% trans "Positive Exponents" %}

{% trans "A positive exponent means multiplying the base by itself that many times. Example: 2³ = 2 × 2 × 2 = 8." %}

{% trans "Negative Exponents" %}

{% trans "A negative exponent means taking the reciprocal. Example: 2⁻³ = 1/(2³) = 1/8 = 0.125." %}

{% trans "Fractional Exponents" %}

{% trans "A fractional exponent represents a root. Example: 8^(1/3) = ∛8 = 2 (cube root of 8)." %}

{% trans "Roots" %}

{% trans "A root is the inverse of exponentiation. Square root (√) is the 2nd root, cube root (∛) is the 3rd root, etc." %}
