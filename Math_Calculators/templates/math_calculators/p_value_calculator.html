{% trans "What is a P-Value?" %}

{% trans "The p-value is the probability of observing a test statistic as extreme as, or more extreme than, the observed value under the null hypothesis. It helps determine statistical significance." %}

{% trans "Common significance levels: α = 0.05 (5%), α = 0.01 (1%), α = 0.10 (10%)" %}

{% trans "Test Types" %}

  • {% trans "Z-Test" %}: {% trans "Used when population standard deviation is known" %}
  • {% trans "T-Test" %}: {% trans "Used when population standard deviation is unknown (requires degrees of freedom)" %}
  • {% trans "Chi-Square Test" %}: {% trans "Used for testing independence or goodness of fit" %}
  • {% trans "F-Test" %}: {% trans "Used for comparing variances or in ANOVA" %}

{% trans "Interpreting P-Values" %}

  • p < 0.01: {% trans "Very strong evidence against null hypothesis" %}
  • p < 0.05: {% trans "Strong evidence against null hypothesis (statistically significant)" %}
  • p < 0.10: {% trans "Weak evidence against null hypothesis" %}
  • p ≥ 0.10: {% trans "Little to no evidence against null hypothesis" %}
