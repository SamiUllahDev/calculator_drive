{% trans "Percent error measures the accuracy of a measurement by comparing the experimental (measured) value to the theoretical (expected) value. It shows how far off a measurement is from the true value as a percentage." %}

  • {% trans "Relative Error" %}: {% trans "The ratio of absolute error to the absolute value of the theoretical value (unitless)" %}
