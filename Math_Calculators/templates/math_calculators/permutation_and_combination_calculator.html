{% trans "What are Permutations and Combinations?" %}

{% trans "Permutations and combinations are ways to count arrangements and selections of items from a set." %}

  • {% trans "Permutation P(n,r)" %}: {% trans "Number of ways to arrange r items from n items where order matters" %}
  • {% trans "Combination C(n,r)" %}: {% trans "Number of ways to select r items from n items where order does not matter" %}

{% trans "Formulas" %}

{% trans "Permutation Formula" %}

P(n, r) = n! / (n - r)!

{% trans "Example: P(5, 3) = 5! / (5-3)! = 120 / 2 = 60" %}

{% trans "Combination Formula" %}

C(n, r) = n! / (r! × (n - r)!)

{% trans "Example: C(5, 3) = 5! / (3! × 2!) = 120 / 12 = 10" %}

{% trans "Key Differences" %}

  • {% trans "Permutations" %}: {% trans "Order matters (ABC is different from CBA)" %}
  • {% trans "Combinations" %}: {% trans "Order does not matter (ABC is the same as CBA)" %}
  • {% trans "Permutations are always greater than or equal to combinations for the same n and r" %}
