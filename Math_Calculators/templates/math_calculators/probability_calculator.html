{% trans "What is Probability?" %}

{% trans "Probability is a measure of the likelihood that an event will occur. It ranges from 0 (impossible) to 1 (certain), or 0% to 100%." %}

{% trans "Probability Types" %}

  • {% trans "Basic Probability P(A)" %}: {% trans "P(A) = Favorable outcomes / Total outcomes" %}
  • {% trans "Conditional Probability P(A|B)" %}: {% trans "Probability of A given B: P(A|B) = P(A and B) / P(B)" %}
  • {% trans "Joint Probability P(A and B)" %}: {% trans "Probability of both A and B: P(A and B) = P(A) × P(B) for independent events" %}
  • {% trans "Union Probability P(A or B)" %}: {% trans "Probability of A or B (or both): P(A or B) = P(A) + P(B) - P(A and B)" %}

{% trans "Key Concepts" %}

  • {% trans "Independent Events" %}: {% trans "Events where the occurrence of one does not affect the other" %}
  • {% trans "Dependent Events" %}: {% trans "Events where the occurrence of one affects the probability of the other" %}
  • {% trans "Mutually Exclusive" %}: {% trans "Events that cannot occur simultaneously (P(A and B) = 0)" %}

{% trans "Examples" %}

{% trans "Example 1: Basic Probability" %}

{% trans "Problem" %}: {% trans "Rolling a 6 on a die" %}

{% trans "Favorable" %}: 1

{% trans "Total" %}: 6

{% trans "Result" %}: P(6) = 1/6 ≈ 0.1667 or 16.67%

{% trans "Example 2: Joint Probability" %}

{% trans "Problem" %}: {% trans "Rolling two 6s (independent events)" %}

{% trans "P(6 on first)" %}: 1/6

{% trans "P(6 on second)" %}: 1/6

{% trans "Result" %}: P(6 and 6) = (1/6) × (1/6) = 1/36 ≈ 0.0278
