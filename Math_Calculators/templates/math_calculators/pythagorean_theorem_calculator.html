{% trans "What is the Pythagorean Theorem?" %}

{% trans "The Pythagorean theorem states that in a right triangle, the square of the hypotenuse (c) equals the sum of squares of the other two sides (a and b)." %}

a² + b² = c²

{% trans "Calculation Types" %}

  • {% trans "Find Hypotenuse (c)" %}: {% trans "c = √(a² + b²)" %}
  • {% trans "Find Side a" %}: {% trans "a = √(c² - b²)" %}
  • {% trans "Find Side b" %}: {% trans "b = √(c² - a²)" %}

{% trans "Key Points" %}

  • {% trans "The theorem only applies to right triangles (triangles with a 90° angle)" %}
  • {% trans "The hypotenuse (c) is always the longest side" %}
  • {% trans "The hypotenuse is opposite the right angle" %}
  • {% trans "Sides a and b are the legs of the right triangle" %}

{% trans "Examples" %}

{% trans "Example 1: Find Hypotenuse" %}

{% trans "Given" %}: a = 3, b = 4

{% trans "Calculate" %}: c = √(3² + 4²) = √(9 + 16) = √25

{% trans "Result" %}: c = 5

{% trans "Example 2: Find Side" %}

{% trans "Given" %}: b = 4, c = 5

{% trans "Calculate" %}: a = √(5² - 4²) = √(25 - 16) = √9

{% trans "Result" %}: a = 3
