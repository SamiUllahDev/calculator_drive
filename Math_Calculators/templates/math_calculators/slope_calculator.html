{% trans "Slope Formula" %}

{% trans "The slope (m) of a line passing through two points (x₁, y₁) and (x₂, y₂) is:" %}

m = (y₂ - y₁) / (x₂ - x₁)

{% trans "Line Equation Forms" %}

  • {% trans "Slope-Intercept" %}: y = mx + b
  • {% trans "Point-Slope" %}: y - y₁ = m(x - x₁)
  • {% trans "Standard" %}: Ax + By = C

{% trans "Examples" %}

{% trans "Example 1: From Two Points" %}

{% trans "Points" %}: (2, 3) and (5, 7)

{% trans "Slope" %}: m = (7 - 3) / (5 - 2) = 4/3

{% trans "Equation" %}: y = (4/3)x + 1/3
