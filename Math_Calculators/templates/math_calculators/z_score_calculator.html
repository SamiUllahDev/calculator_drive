{% trans "Z-Score Formula" %}

z = (X - μ) / σ

  • z: {% trans "Z-score (standard score)" %}
  • X: {% trans "Raw score (observed value)" %}
  • μ: {% trans "Mean (average)" %}
  • σ: {% trans "Standard deviation" %}

{% trans "Interpretation" %}

  • {% trans "z = 0: Value equals the mean" %}
  • {% trans "z > 0: Value is above the mean" %}
  • {% trans "z < 0: Value is below the mean" %}
  • {% trans "|z| < 1: Within 1 standard deviation (68% of data)" %}
  • {% trans "|z| < 2: Within 2 standard deviations (95% of data)" %}
  • {% trans "|z| < 3: Within 3 standard deviations (99.7% of data)" %}
