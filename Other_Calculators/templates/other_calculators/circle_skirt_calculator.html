{% trans "How Circle Skirts Work" %}

{% blocktrans trimmed %}A circle skirt is cut from a circular piece of fabric with a hole for the waist. The waist radius determines the size of the inner circle, and the outer radius includes the desired skirt length plus hem allowance. Different fractions of a circle create different levels of flare and fullness. Need help converting between inches and centimeters? Use our{% endblocktrans %} {% trans "Conversion Calculator" %}.

R{% trans "waist" %} = {% trans "Waist" %} / (2 × π × {% trans "fraction" %})

{% trans "fraction: 1 = full, 0.75 = ¾, 0.5 = half, 0.25 = quarter" %}

{% trans "4 Skirt Types" %}

{% trans "Full Circle (360°)" %}

{% trans "Maximum flare and twirl. Uses most fabric. Great for dance skirts and formal wear." %}

{% trans "¾ Circle (270°)" %}

{% trans "Beautiful drape with less bulk. Requires one seam. Nice compromise between fullness and practicality." %}

{% trans "Half Circle (180°)" %}

{% trans "Moderate flare, very wearable. Uses about half the fabric of a full circle. Perfect for everyday." %}

{% trans "¼ Circle / A-Line (90°)" %}

{% trans "Minimal flare, classic A-line. Most fabric-efficient and easy to sew. Needs two panels." %}

{% trans "Sewing Tips" %}

  • {% trans "Always cut on the bias (diagonal grain) for the best drape." %}
  • {% trans "Let the skirt hang for 24 hours before hemming — bias-cut fabric stretches." %}
  • {% trans "Use a narrow hem (rolled or baby hem) for the smoothest finish on curved edges." %}
  • {% trans "Add 1-2 inches to length if using stretchy fabric." %}
  • {% trans "Add a separate straight-grain waistband rather than folding the circle fabric." %}

{% trans "Important Notes" %}

  • {% trans "Seam allowance is added to the waist circumference before calculating the radius." %}
  • {% trans "Hem allowance is added to the skirt length (outer radius)." %}
  • {% trans "Fabric width affects panel count and cutting layout." %}
