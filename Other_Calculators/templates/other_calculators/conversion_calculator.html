{% trans "Frequently Asked Questions" %}

{% trans "How accurate are the conversions?" %}

{% trans "All conversions use internationally recognized conversion factors and are computed using high-precision numerical libraries. Results are scientifically accurate for educational and practical use." %}

{% trans "Why is temperature conversion different from other conversions?" %}

{% trans "Most unit conversions use simple multiplication factors. Temperature scales (Celsius, Fahrenheit, Kelvin, Rankine) use offset-based formulas because their zero points differ. For example, °F = (°C × 9/5) + 32 involves both multiplication and addition." %}

{% trans "What is the difference between KB and KiB?" %}

{% trans "KB (Kilobyte) uses the SI standard where 1 KB = 1,000 bytes. KiB (Kibibyte) uses the binary standard where 1 KiB = 1,024 bytes. The binary prefixes (Ki, Mi, Gi) were introduced by the IEC to remove ambiguity in digital storage measurements." %}

{% trans "Can I convert between categories?" %}

{% trans "No, conversions are only possible within the same category. For example, you can convert meters to feet (both length), but not meters to kilograms (length vs. weight). Select a category first, then choose units within it." %}

{% trans "How does the swap button work?" %}

{% trans "The swap button (↕) exchanges the 'From' and 'To' units instantly. This is useful when you want to perform the reverse conversion without manually reselecting units." %}
