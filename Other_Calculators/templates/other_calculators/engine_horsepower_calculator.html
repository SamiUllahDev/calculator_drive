{% trans "What is Engine Horsepower?" %}

{% blocktrans trimmed %}Engine horsepower measures how much power an engine produces. It is calculated from torque (rotational force) and RPM (engine speed) using the formula HP = (Torque × RPM) / 5252. This is the standard formula used for dyno testing, engine tuning, and comparing vehicle performance across different makes and models. For general-purpose power calculations beyond engines, the{% endblocktrans %} {% trans "Horsepower Calculator" %} {% trans "covers a wider range of scenarios." %}

{% trans "Core Formulas" %}

{% trans "Horsepower" %}: HP = (Torque × RPM) / 5252

{% trans "Torque" %}: Torque = (HP × 5252) / RPM

{% trans "RPM" %}: RPM = (HP × 5252) / Torque
