{% trans "What is Heat Index?" %}

{% trans "The Heat Index (also known as 'apparent temperature' or 'feels like temperature') is a measure of how hot it feels when relative humidity is factored in with the actual air temperature. It is an important indicator for heat-related health risks, especially during hot weather. The heat index is calculated using the Rothfusz equation, which takes into account both temperature and relative humidity to determine the perceived temperature." %}

{% trans "Heat Index Formula (Rothfusz Equation)" %}

{% trans "Main Formula:" %}

HI = -42.379 + 2.04901523×T + 10.14333127×RH - 0.22475541×T×RH

- 6.83783×10⁻³×T² - 5.481717×10⁻²×RH² + 1.22874×10⁻³×T²×RH

+ 8.5282×10⁻⁴×T×RH² - 1.99×10⁻⁶×T²×RH²

{% trans "Where:" %}

  • {% trans "HI = Heat Index (°F)" %}
  • {% trans "T = Temperature (°F)" %}
  • {% trans "RH = Relative Humidity (%)" %}

{% trans "Adjustments:" %}

  • {% trans "If RH < 13% and 80°F ≤ T ≤ 112°F: Subtract adjustment" %}
  • {% trans "If RH > 85% and 80°F ≤ T ≤ 87°F: Add adjustment" %}
