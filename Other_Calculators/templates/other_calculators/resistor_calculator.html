{% trans "What is a Resistor?" %}

{% blocktrans %}A resistor is an electrical component that limits or controls the flow of electric current in a circuit. Resistance is measured in ohms (Ω) and is usually shown by colored bands on the resistor body. This tool helps you decode those color bands, combine resistors in series or parallel, and calculate power dissipation — all for free.{% endblocktrans %}

{% trans "All Formulas at a Glance" %}

{% trans "Color Code" %}: {% trans "R = (digits) × multiplier ± tolerance%" %}

{% trans "Series" %}: R_total = R1 + R2 + R3 + …

{% trans "Parallel" %}: 1/R_total = 1/R1 + 1/R2 + 1/R3 + …

{% trans "Power" %}: P = V² / R  |  I = V / R
