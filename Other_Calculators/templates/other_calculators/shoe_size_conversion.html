{% trans "5 Sizing Systems Supported" %}

{% trans "US (United States)" %}

{% trans "The most common system in North America. Men's sizes 4-16, Women's sizes 4-14. Half sizes are common. Women's sizes are typically 1.5-2 numbers higher than men's for the same foot length." %}

{% trans "UK (United Kingdom)" %}

{% trans "Used in the UK and some Commonwealth countries. Typically about 1 size smaller than US. Men's sizes 3-15, Women's sizes 3-13. Half sizes are available." %}

{% trans "EU (Europe)" %}

{% trans "The Paris point system used across continental Europe. Sizes range from 35-50. Unisex — the same number applies regardless of gender. Whole sizes only (no half sizes)." %}

{% trans "CM (Centimeters)" %}

{% trans "Your actual foot length in centimeters, measured from heel to longest toe. The universal measurement that all other systems convert through. Range: 20-35 cm." %}

{% trans "JP (Japan)" %}

{% trans "Japanese sizes are identical to centimeters — they represent the foot length in cm. Common in Japan and some other Asian countries." %}

{% trans "Conversion Formulas" %}

{% trans "US Men's ↔ CM:" %}

CM = (US + 22) × 0.847  |  US = (CM ÷ 0.847) - 22

{% trans "US Women's ↔ CM:" %}

CM = (US + 20.5) × 0.847  |  US = (CM ÷ 0.847) - 20.5

{% trans "UK Men's ↔ CM:" %}

CM = (UK + 23) × 0.847  |  UK = (CM ÷ 0.847) - 23

{% trans "UK Women's ↔ CM:" %}

CM = (UK + 21.5) × 0.847  |  UK = (CM ÷ 0.847) - 21.5

{% trans "EU ↔ CM:" %}

CM = (EU - 2) ÷ 1.5 × 2.54  |  EU = (CM ÷ 2.54) × 1.5 + 2
