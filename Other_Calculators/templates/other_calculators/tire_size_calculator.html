{% trans "Understanding Tire Sizes" %}

{% blocktrans %}Tire sizes describe the physical dimensions of a tire — width, aspect ratio (sidewall height as a % of width), and rim diameter. These numbers affect ride comfort, handling, speedometer accuracy, and fuel economy. This calculator works with both metric (P-metric) and imperial (flotation) formats.{% endblocktrans %}

{% trans "Tire Size Formats" %}

{% trans "Metric (P-Metric)" %}

205/55R16

  • 205 — {% trans "Width in mm" %}
  • 55 — {% trans "Aspect ratio (sidewall = 55% of width)" %}
  • R — {% trans "Radial construction" %}
  • 16 — {% trans "Rim diameter in inches" %}

{% trans "Imperial (Flotation)" %}

31x10.5R15

  • 31 — {% trans "Overall diameter in inches" %}
  • 10.5 — {% trans "Width in inches" %}
  • R — {% trans "Radial construction" %}
  • 15 — {% trans "Rim diameter in inches" %}

{% trans "Key Formulas" %}

{% trans "Sidewall" %} = ({% trans "Width" %} × {% trans "Aspect Ratio" %} / 100) / 25.4

{% trans "Diameter" %} = {% trans "Rim" %} + 2 × {% trans "Sidewall" %}

{% trans "Circumference" %} = π × {% trans "Diameter" %}

{% trans "Rev/mile" %} = 63,360 / {% trans "Circumference" %}

{% trans "Actual Speed" %} = {% trans "Indicated" %} × ({% trans "New Dia" %} / {% trans "Old Dia" %})
