{% trans "Wind Chill Index Formula" %}

WC = 35.74 + 0.6215×T - 35.75×V^0.16 + 0.4275×T×V^0.16

{% trans "Where" %}:

WC = Wind Chill (°F)

T = Air Temperature (°F)

V = Wind Speed (mph)

{% trans "Applicability" %}

{% trans "Wind chill applies when:" %}

• {% trans "Temperature ≤ 50°F (10°C)" %}

• {% trans "Wind speed ≥ 3 mph (4.8 km/h)" %}

{% trans "Below these thresholds, wind chill equals air temperature." %}

{% trans "Risk Categories" %}

≥ 32°F (0°C):
